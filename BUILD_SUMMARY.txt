================================================================================
                            HOW TO USE
================================================================================

DEVELOPMENT:
  $ pnpm install
  $ pnpm dev
  > Open http://localhost:3000

CUSTOMIZE:
  1. Colors: Edit globals.css CSS custom properties
  2. Products: Update lib/product-data.ts
  3. Content: Edit components directly
  4. Images: Add to public/images/

DEPLOY TO VERCEL:
  1. Push code to GitHub
  2. Connect repo to Vercel
  3. Click "Deploy"
  4. Done! Auto-deployed on push
